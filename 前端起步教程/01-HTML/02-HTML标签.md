## 块标签

1.霸占一行，不能与其他任何元素并列；

2.能接受宽、高。如果不设置宽度，那么宽度将默认变为父亲的100%。

- 标题标签 `h1~h6`
- div p(段落标签)
- ul li
- hr br
- table tr td
- form
- select option
- input textarea
- video audio

## 行内标签

1.与其他行内元素并排；

2.不能设置宽、高。默认的宽度，就是文字的宽度。

- span 
- a
- 上标<sup> 下标<sub>

## 行内块标签

- img(src alt width height)

## 行和块直接的转换

```css
div{
  display: inline;
}
```
