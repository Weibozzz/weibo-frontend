## Git

git 是什么？一步一步来，后面再给你慢慢解开它神秘的面纱，先跟着用

## 安装

进入 <https://git-scm.com/downloads> 选择平台安装

## 克隆代码拉代码
### 第一次使用先克隆代码到本地

```shell
git clone https://github.com/Weibozzz/weibo-frontend.git
```
### 拉取代码
进入代码文件夹

```shell
cd weibo-frontend
git pull
```
## .gitignore 文件
也就是忽略 `git` 追踪的一些文件，常用于项目自动生成的不属于我们
的开发代码。如果不添加我们的 `git` 记录会比较不干净
