## 函数
函数：就是将一些功能或语句进行**封装**，
在需要的时候，通过**调用**的形式，执行这些语句。

- 将大量重复的语句抽取出来，写在函数里，以后需要这些语句的时候，可以直接调用函数，避免重复劳动。
- 简化编程，让编程模块化。高内聚、低耦合。

## 函数定义

```js
// 函数声明
function fun1(a, b){
	return a+b;
}
// 函数表达式
var fun2 = function() {
  console.log("我是匿名函数中封装的代码");
};
```
## 箭头函数

```js
var fn = () => {}
```

## 函数参数和返回值

```js
function fun1(a, b){ // 形参
	return a+b; // 有返回值
}
fun1(1,3) // 实参
```
